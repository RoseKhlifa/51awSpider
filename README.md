# 51aw 吃瓜网文章爬取工具

> 一个基于 Python + BeautifulSoup 的网页爬虫，用于批量爬取 **51aw 吃瓜网** 的搜索结果文章、标题、链接及正文简介，帮助用户快速筛选感兴趣的内容。

---
![程序运行截图](images/demo.png)
## ✨ 项目简介

本项目通过模拟浏览器访问，对 **51aw 吃瓜网搜索页面**进行解析，自动完成以下工作：

- 根据关键词搜索文章
- 自动分页遍历搜索结果
- 提取每篇文章的：
  - 标题
  - 文章链接
  - 正文简介（已过滤无关内容）
- 支持设置**最大抓取文章数量（limit）**
- 适合用于内容筛选、学习爬虫结构分析

---

## 🧩 功能特性

- ✅ 关键词搜索
- ✅ 自动分页处理
- ✅ 可设置文章抓取上限
- ✅ 自动过滤：
  - `blockquote` 引用内容
  - 图片标签（避免 `alt/title` 干扰）
  - 无效 / 空段落
- ✅ 支持中途终止正文提取（基于文本标志）
- ✅ 程序结构清晰，便于二次开发

---
